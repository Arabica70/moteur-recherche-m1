Sujet de TP: FORUM

Exercice 1: Inscription et connexion

  • Connectez vous sur phpmyadmin et éxecutez le script suivant
  • Créez une page HTML contenant un formulaire contenant trois champs:
    1. Le nom de l'utilisateur,
    2. Son mot de passe,
    3. La vérification du mot de passe.
  • Le formulaire appelle un script qui vérifie que les trois champs sont bien remplis, que le mot de passe et sa vérification son les memes. Si c'est le cas, on insère l'utilisateur et son mot de passe dans la base de données, dans la table Utilisateurs, puis affiche une page de connexion. Sinon, le script renvoie faux.
  • Ecrire une page de connexion à l'aide d'un formulaire, qui appelle un script php qui vérifie que le login et le mot de passe apparait dans la base de données. Si c'est le cas, on initialise le champ "user" dans la superglobale $_SESSION. Sinon on renvoie faux.

Exercice 2: forum

Créer une page forum.php dans laquelle on place:

  • Un formulaire permettant d'ajouter un nouveau sujet.
  • La liste des sujet déjà crée. Pour chacun dans sujet, il lien permet de rapeller la page forum.php avec la méthode get et le nom du sujet.
  • Si le champ 'sujet' existe dans la superglobale $_GET, au lieu d'afficher le formulaire et la liste précédement évoqués, on affiche la liste des messages sur ce sujet. De plus, on ajoute un formulaire permettant d'ajouter un message dans la base de données.
