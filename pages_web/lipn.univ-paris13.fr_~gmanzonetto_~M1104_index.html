Emplois du temps

Séances du Cours:
1) 19 nov. Généralités, Unitées utilisées, mesures de performance, classification des réseaux. Topologie de réseaux: physique, logique, logicielle.
2) 28 nov. Normes: organismes compétents. Logiciels de réseaux: structuration en couches, protocole, pile de protocoles, interface, services. Encapsulation, PDU, SDU, PCI, Fragmentation, service avec/sans connexion, service confirmé/non confirmé, les sept couches du Modèle ISO-OSI (noms et fonctionnalités).
3) 4 dec. Modèle TCP/IP, Couche Hôte-Réseau: adresses MAC, Ethernet, Token Ring, Couche Internet, Protocole IP, Adresses IP (classes).
4) 6 dec. Protocole ARP, Protocole ICMP, Couche Transport: TCP, UDP. Couche Application, préliminaires sur les codes.
5) 8 janvier. Code VRC pair/impair, LRC pair/impair, VRC+LRC pair/impair, code de Hamming, code polynômial.
