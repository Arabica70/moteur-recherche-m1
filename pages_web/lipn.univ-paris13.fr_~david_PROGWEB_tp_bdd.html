Sujet de TP: PHP, PDO et les bases de données

Exercice: accès à la base de données

  • Téléchargez le fichier TP_BLOG.sql
  • Connectez vous à l'adresse suivante. (login: "E" puis votre numéro étudiant, mot de passe: INE inscrit sur votre carte d'étudiant). Importez le fichier TP_BLOG.sql
  • Modifiez le constructeur de votre classe afin qu'il contiennent une connexion à la base de donnée, en utilisant la classe PDO. Le nom du serveur est bien entendu marseille. Le nom de votre base de donnée est "BDE" suivi de votre numéro étudiant.
  • Modifier les méthodes de la classe afin que le code HTML soit engendré non plus à partir de tableaux mais à partir du contenu de la base de donnée.
