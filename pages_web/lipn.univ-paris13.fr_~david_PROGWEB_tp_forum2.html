Sujet de TP: FORUM, les émoticons

Exercice 1: Créer une nouvelle table dans la base de données

  • Téléchargez l'archive Emoticon.tgz et décompressez la dans le même répertoire que le TP précédent.
  • Connectez vous à phpmyadmin et créez un nouvelle table Emoticon dans votre base de données contenant:
    1. L'url d'une emoticon (texte),
    2. La chaîne de caratères correspondant à l'émoticon.
    3. insérez plusieurs émoticons dans la base de données

Exercice 2: la classe Emoticon

Créer un fichier Emoticon.php dans lequel on définit la classe Emoticon:

  • Son constructeur se connecte à la base de données et initialise un tableau images dont les clés sont les chaînes de caractères correspondant à des émoticons et les valeurs sont les urls.
  • Ecrire la méthode filtre qui prend en entrée un texte et renvoie un texte modifié, dans lequel toutes les chaînes de caractères correspondant à des émoticons sont remplacées par des balises img contenant l'url de l'image.
  • Utilisez la classe Emoticon et sa méthode filtre pour charger les messages du forum de telle sorte que les émoticons apparaissent sous forme d'images de même taille.
