Sujet de TP: Javascript et Jquery

Exercice: Affichage

  • Affichez un pop-up contenant la phrase "Hello World" avec la commande alert
  • Insérer dans une page Web la phrase "Hello World", lorsque l'utilisateur clique sur un bouton.
  • Ecrire la fonction factorielle et afficher le résultat de 4! dans la console.

Exercice: DOM et JQuery

Reprendre l'exercice du formulaire d'inscription en HTML.
  • Rendre des champs obligatoires (required dans le code HTML)
  • Si ce n'est pas déjà fait, écrire dans le code html deux champs pour choix du mot de passe de l'utilisateur (le deuxième servant à la vérification). Lorsque le deuxième champ est modifié, vérifiez que le mot de passe saisi est égal à celui du premier champ. S'ils sont différents, insérez le message "mot de passe incorrect" à coté du deuxième champ. Pour cela, on utilisera du code javascript qui insère un champ de type <span> à coté de la balise mot de passe.

Exercice: Menu Mac OS

Modifier le menu de l'exercice 2 du TP 3 afin que lorsque l'utilisateur fait passer son curseur sur un bouton, les voisins les plus proches prennent une hauteur et une largeur de 62 pixels.

Exercice: Gallerie Photo

Modifier le menu de l'exercice 3 du TP 3 afin de créer une gallerie de tableaux. Lorsque l'utilisateur clique sur une image, une fenetre apparait dans laquelle l'image s'affiche en grand, avec deux boutons (précédents et suivants). En cliquant sur l'un des deux boutons, l'utilisateur doit pouvoir passer à l'image suivante ou précédente dans la gallerie.
