Sujet de TP: PHP et les bases de données

Exercice 1: le premier script

  • Créez un script test.php dans le répertoire ~/public_html/php/ qu'il génère une page HTML5 minimale affichant le numéro de version de l’interpréteur PHP utilisée.
  • Vérifiez avec un navigateur qu'il est bien exécuté avec l'URL http://marseille/~"numero étudiant"/test.php.

Exercice 2: formulaire et PHP

Utilisez la méthode POST et le formulaire décrit dans inscription.html pour ajouter un utilisateur dans la base de données. Le code PHP sera inscrit dans un répertoire ~/public_html/php/ dans un fichier inscription.php.

  • Utiliser un filtre pour savoir si l'adresse mail est bien au bon format
  • Vérifier que les champs ne sont pas vides.
  • Initialiser un tableau donnees aux m\^emes clés et valeurs que le tableau $_POST
  • Afficher une nouvelle page produite par PHP, contenant une table HTML qui énumère les clés et les valeurs de données.

Exercice 3: le blog en PHP

  • Reprendre le TP noté
  • Ecrire dans un fichier article_blog.php une classe dont le constructeur initialise les tableaux de 3 cases liens, contenu, titre et image aux valeurs que vous extrairez de votre page HTML. Vous supprimerez ensuite les articles de votre page HTML.
  • Ecrire dans la classe une méthode ecrireArticle qui prend en entrée un entier, qui représente la clé que vous utiliserez pour accéder aux quatres tableaux et produire en PHP un article HTML à l'aide de ces valeurs.
  • écrire dans la classe une méthode ecrireContenu, qui utilise ecrireArticle pour afficher les 3 articles.
  • Renommer votre fichier index.html en index.php. Utilisez un script PHP pour inclure article_blog.php, puis utilisez la classe pour produire les 3 articles.
