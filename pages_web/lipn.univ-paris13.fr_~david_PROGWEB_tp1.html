Sujet de TP: les balises HTML5

Exercice 0: Mise en place

  • A la racine de votre compte, s'il n'existe pas déjà, créez le répertoire public_html.
  • Entrez les commandes suivantes:
    • cd
    • chmod -R 700 *
    • chmod 755 .
    • chmod 755 public_html
    • chmod -R 755 public_html/*
  • Dans le répertoire public_html, créez un fichier index.html contenant le code: <html> Je suis une page de test. </html>
  • Ouvrez un navigateur. Entrez l'url suivante: http://marseille/~VotreNumeroEtudiant/index.html
Pour la suite des exercices, en plus d'utiliser le navigateur pour visualiser vos pages, vous testerez votre code avec le validateur W3C.

Exercice 1: la première page

Créez une page index.html contenant les informations suivantes:
  • Titre : Bienvenue sur ma première page.
  • Encodage : utf-8.
  • en-tête: réecrire le titre de la page,
  • créez une barre de navigation contenant un lien vers la page inscription.html (on créera ce fichier dans l'exercice suivant),
  • créez un article (par étudiant devant machine) contenant:
    • en-tête: vos noms et prénoms,
    • dans une section: des informations le concernant,
      • sous forme de liste, vos numéros étudiants et votre grade universitaire.
      • dans une balise aside, vos photos d'indentité (ou une autre image).
  • créer un article contenant:
    • en en-tête: "Média",
    • une vidéo (format mp4, webm ou ogv) et un mp3 de votre choix,
  • créer un article (par étudiant devant machine) contenant un tableau de vos notes dans les différentes matières du premier semestre. Vous utiliserez les balises thead et tbody.
  • écrire le nom des auteurs en pied de page.
Dans un premier temps, on se contente de placer l'information dans les bonnes balises, sans se soucier de l'affichage à l'écran.
Placez les fichiers qui ne sont pas au format html (Ex: images, son, ...) dans d'autres répertoires (Ex: media/)
Afin que la liste du contenu de ces répertoires ne soient pas accessibles à l'aide d'un navigateur, placez un fichier index.html vide dans ces répertoires.

Exercice 2: formulaire

Créez une nouvelle page contenant un formulaire utilisant les types proposés par html5, permettant à l'utilisateur d'entrer:
  • un nom,
  • un prénom,
  • un pseudonyme (Ce champ est obligatoire),
  • un mot de passe (ce que l'utilisateur entre ne doit pas être visible) (Ce champ est obligatoire),
  • une adresse mail (Ce champ est obligatoire),
  • son genre,
  • date de naissance,
  • lieu de naisssance,
  • sa situation (célibataire, en couple, pacsé, marié),
  • son numéro de téléphone,
  • Un bouton de type submit contenant le texte "Je m'inscris!".
